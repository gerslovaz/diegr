# Package index

## Visualization

Functions for visualization of HD-EEG data

- [`animate_scalp()`](https://gerslovaz.github.io/diegr/reference/animate_scalp.md)
  : 3D scalp plot animation in time
- [`animate_topo()`](https://gerslovaz.github.io/diegr/reference/animate_topo.md)
  : Topographic map animation in time
- [`animate_topo_mean()`](https://gerslovaz.github.io/diegr/reference/animate_topo_mean.md)
  : Animate EEG average topographic map with confidence bounds
- [`boxplot_epoch()`](https://gerslovaz.github.io/diegr/reference/boxplot_epoch.md)
  : Plot interactive boxplots of EEG amplitude on epoch level
- [`boxplot_subject()`](https://gerslovaz.github.io/diegr/reference/boxplot_subject.md)
  : Plot interactive boxplots of EEG amplitude across subjects
- [`boxplot_rt()`](https://gerslovaz.github.io/diegr/reference/boxplot_rt.md)
  : Plot interactive boxplots of response times
- [`interactive_waveforms()`](https://gerslovaz.github.io/diegr/reference/interactive_waveforms.md)
  : Plot interactive waveform graph
- [`plot_point_mesh()`](https://gerslovaz.github.io/diegr/reference/plot_point_mesh.md)
  : Plot point mesh
- [`plot_time_mean()`](https://gerslovaz.github.io/diegr/reference/plot_time_mean.md)
  : Plot time curve of average EEG signal with confidence interval
- [`plot_topo_mean()`](https://gerslovaz.github.io/diegr/reference/plot_topo_mean.md)
  : Plot topographic map of average EEG signal
- [`scalp_plot()`](https://gerslovaz.github.io/diegr/reference/scalp_plot.md)
  : Plot scalp map of EEG signal
- [`topo_plot()`](https://gerslovaz.github.io/diegr/reference/topo_plot.md)
  : Plot topographic map of EEG signal

## Computational and Utility Functions

Functions supporting HD-EEG data processing, analysis and visualization

- [`baseline_correction()`](https://gerslovaz.github.io/diegr/reference/baseline_correction.md)
  : Baseline correction
- [`compute_mean()`](https://gerslovaz.github.io/diegr/reference/compute_mean.md)
  : Calculate mean in temporal or spatial domain
- [`create_scale()`](https://gerslovaz.github.io/diegr/reference/create_scale.md)
  : Create colour scale used in topographic figures
- [`make_triangulation()`](https://gerslovaz.github.io/diegr/reference/make_triangulation.md)
  : Make triangulation of 2D point mesh
- [`outliers_epoch()`](https://gerslovaz.github.io/diegr/reference/outliers_epoch.md)
  : Select outlier epochs
- [`pick_data()`](https://gerslovaz.github.io/diegr/reference/pick_data.md)
  : Subsets EEG data by group, subject, sensor, time, experimental
  condition or epoch
- [`pick_region()`](https://gerslovaz.github.io/diegr/reference/pick_region.md)
  : Choose region of interest
- [`point_mesh()`](https://gerslovaz.github.io/diegr/reference/point_mesh.md)
  : Create regular mesh of points
- [`summary_stats_rt()`](https://gerslovaz.github.io/diegr/reference/summary_stats_rt.md)
  : Compute summary statistics of reaction times
